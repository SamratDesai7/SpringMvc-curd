# Spring MVC CRUD Application

A simple and structured **Spring MVC CRUD web application** built using Java.  
This project demonstrates how to perform **Create, Read, Update, and Delete** operations using the Spring MVC architecture with JSP views and a relational database.

It is ideal for beginners who want hands-on experience with Spring MVC and understand how a real Java web application is structured.
